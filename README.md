### `FRONTEND`
* Navigate to the "frontend" folder <br>
* Install all dependencies. 
> npm install
* Run from command prompt
> npm start
* Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `APIs`

#### `AUCTIONS`
--------------------------------
##### GET
http://localhost:9000/api/auctions

http://localhost:9000/api/auctions
